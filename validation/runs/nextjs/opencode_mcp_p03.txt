### Risk Profile

This file defines:
- The `BaseServer` class (50+ methods including `handleRequest`, `render`, `pipe`, `prepare`)
- Key types (`RequestLifecycleOpts`, `MiddlewareRoutingItem`, `NextEnabledDirectories`) consumed across server, export, and build subsystems

**Modifying public API methods or types** on `BaseServer` will break the two subclass servers (`NextNodeServer`, `DevServer`) and any file consuming those types. **Modifying internal implementation details** primarily affects webpack build plugins, client error handling, and the incremental cache — the ~2200+ impacted units across the codebase.

**To verify changes**: run `pnpm --filter=next types` first, then the server integration test suites (`test/development/`, `test/e2e/app-dir/`).
