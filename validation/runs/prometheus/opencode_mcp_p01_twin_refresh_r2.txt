## What This Project Is

**Prometheus** is a CNCF-graduated systems & service monitoring system. It pulls metrics from targets via HTTP, stores them in a local TSDB, evaluates PromQL rules, and triggers alerts. Two run modes exist: **server** (full TSDB with query/rules/alerting) and **agent** (WAL-only, forward-only).

**VERIFIED** — `README.md:22-24` ("Prometheus...is a systems and service monitoring system"), `cmd/prometheus/main.go:687-690` (agent vs server storage path logic), `VERSION:1` ("3.11.3").

**Core data flow** (from `cmd/prometheus/main.go:892-1003`):
```
SD Discovery -> Scrape Manager -> Fanout Storage (TSDB primary + Remote Write secondary) -> PromQL Engine -> Rule Manager -> Notifier -> Alertmanager
```
