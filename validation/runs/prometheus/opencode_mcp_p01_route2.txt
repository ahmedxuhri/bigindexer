## What This Project Is

**VERIFIED** — This is the [Prometheus](https://prometheus.io) monitoring system, a CNCF-graduated project. It collects metrics via HTTP pull, stores them as time series in a local TSDB, evaluates PromQL queries, and triggers alerts.

- **`VERSION`**: `3.11.3`
- **Module**: `github.com/prometheus/prometheus` (`go.mod:3`)
- **Language**: Go (1.25), 439 source files + 265 test files (345k LOC)
- **Entry points**: `cmd/prometheus/main.go` (server), `cmd/promtool/main.go` (CLI tooling)
