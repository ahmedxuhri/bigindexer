   - Source: `python3 -c "import django"` raises `AttributeError: module 'datetime' has no attribute 'UTC'`
   - **Severity**: HIGH — breaks version detection and any import-time dependency on `django.VERSION`.

2. **Shallow single-commit history** — repo has only 1 commit with no ancestry. This is a shallow clone (likely for analysis), not the full 20-year history. Not a project weakness, but notable for context.
   - Source: `git log --all --oneline` shows only `4d455ae`

3. **No ASGI streaming for request body** — `django/core/handlers/asgi.py:166` has a FIXME: `# FIXME: Allow to override this.`
   - Source: grep result

4. **Async ORM is a sync-to-async wrapper** — related manager methods (`add`, `create`, `remove`, etc.) delegate via `sync_to_async`, meaning they block the event loop thread. No true async database driver integration.
   - Source: `django/db/models/fields/related_descriptors.py:885-1490` — every async method calls `sync_to_async(self.method)`.

5. **GIS Oracle backend has known gaps** — multiple TODOs about missing features (`SDO_DIM_ARRAY` support, geometry field type detection).
